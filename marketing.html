Get Started in Minutes

1

Download

Grab the executable for your platform from the releases page.

Linux x64 macOS Intel macOS ARM64 Windows x64
2

Make Executable

On Linux/macOS, run:

chmod +x Task

Windows executables are ready to run.

3

Install Globally

Optional: Add to your PATH for system-wide access.

sudo mv Task /usr/local/bin/task

Simple Yet Powerful

Basic Commands

Add a new task
task add "Review pull requests"
List all tasks
task list
Mark task as complete
task complete 1
Search for tasks
task search "pull"

Advanced Features

  • JSON Output: Perfect for scripting and integrations
  • Custom Databases: Use different databases for different projects
  • Import/Export: Migrate tasks between systems
  • Plain Text Mode: Clean output for automation
task list --json
