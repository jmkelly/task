Install in Seconds

Run this in your terminal to download and install the latest release:

curl -fsSL https://jmkelly.github.io/task/install.sh | bash

Installs to /usr/local/bin/task (or ~/bin/task if sudo is unavailable). Supports Linux x86_64.

Run this in your terminal — works on Intel and Apple Silicon:

curl -fsSL https://jmkelly.github.io/task/install.sh | bash

Auto-detects Intel (x86_64) or Apple Silicon (arm64). Installs to /usr/local/bin/task.

Open PowerShell and run:

irm https://jmkelly.github.io/task/install.ps1 | iex

Downloads to %LOCALAPPDATA%\Programs\task\task.exe and adds it to your user PATH automatically. Restart your terminal after installing.

Download the binary for your platform directly from the latest release:

🐧 Linux x86_64 🍎 macOS Intel (x86_64) 🍎 macOS Apple Silicon (arm64) 🪟 Windows x86_64 (.exe)
All versions: github.com/jmkelly/task/releases

After downloading on Linux/macOS, make it executable and move to your PATH:
chmod +x task-linux-x64 && sudo mv task-linux-x64 /usr/local/bin/task

Why Choose Task CLI?

🚀

Self-Contained

Single executable includes the .NET runtime and all dependencies. No installation, no prerequisites, no hassle.

🌐

Cross-Platform

Works on Linux, macOS (Intel & Apple Silicon), and Windows. One tool for every machine on your team.

🤖

Agent-Friendly

JSON output, machine-readable errors, and a REST API make Task CLI ideal for LLM agents, automations, and scripts.

⚡

Lightning Fast

Compiled to native code for instant startup. Powered by SQLite with full-text and semantic search.

🖥️

Built-in Kanban board served by the API backend — drag & drop tasks, filter by assignee or tag.

🔍

Powerful Search

Full-text search and semantic (vector) search via the API. Never lose track of your work again.

Simple Yet Powerful

Basic Commands

Start the background server
task server start
Add a new task
task add "Review pull requests" --priority high
List all tasks
task list
Mark a task as complete
task complete a2b3k9
Search tasks
task search "pull request"

Advanced Features

  • JSON Output: Perfect for scripting and agent integrations
  • Kanban Board: Web UI at http://localhost:8080
  • REST API: OpenAPI docs at /scalar
  • Semantic Search: Vector similarity via sqlite-vss
  • Telegram Notifications: Alerts when your task queue empties
  • Import/Export: Migrate tasks from JSON or CSV
